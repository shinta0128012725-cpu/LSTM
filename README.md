# LSTMによる時系列予測モデル（Short-Term Memory）

## 1. 概要

本プロジェクトでは、LSTM（Long Short-Term Memory）を用いて時系列データの将来値を予測するモデルを構築しました。

過去の一定期間のデータから時間的なパターンを学習し、次の時点の値を予測することで、需要予測・株価予測・売上予測などの応用を想定した時系列分析モデルを実装しています。
